Overview
--------
MSI Property Explorer is a Windows desktop tool that allows users to open and search within `.msi` installer packages. It supports searching for specific strings within the "Property" table or across all other tables in the MSI database.

Key Features
------------
- GUI for selecting `.msi` files
- Search for keywords in:
  - Property table (name/value)
  - All other MSI tables (with field-level details)
- List view output with sortable columns
- Search interruption support
- Clear results, close, and user-friendly design

Usage
-----
1. Launch the application.
2. Click "Browse..." to select an MSI file.
3. Choose your search scope: "Properties" or "Tables".
4. Enter a search term and click "Search".
5. Results will appear in the table below.
6. Click "Clear" to remove the results or "Close" to exit.
