# APC (Avarwand Picture Converter)
A **fast, elegant, Python-based GUI tool** that instantly converts **HEIC** and **WebP** photos to **JPEG, PNG, BMP or WebP** – with **100 % quality** and **zero hassle**.

No command line. No quality loss.

---
## Main Features
- **One-click batch conversion** – select 1 or 100 files at once  
- **HEIC → JPEG / PNG / BMP / WebP**  
- **WebP → JPEG / PNG / BMP**  
- **Lossless RGB conversion** (quality=100)  
- **Automatic library installer** – `pillow-heif` is added on first run  
- **Dark-mode UI** with perfect alignment and custom icon  
- **Progress bar** + live status  
- **Fixed 640×480 window** (no resize, no maximize)  
- **Output folder picker** – defaults to Desktop  
- **Clean, modern Tkinter design** – built-in base64 icon  

---
## System Requirements
- **OS**: Windows 10/11 (recommended) – works on Linux/macOS with tiny tweaks  
- **Python**: 3.9+  
- **Dependencies** (auto-installed):  
  - `Pillow`  
  - `pillow-heif`  
  - `tkinter` (built-in)  

---
## How to Use
1. **Run** `Avarwand Picture Converter.py` (or the `.exe`)  
2. Choose **Source Type**: `HEIC` or `WebP`  
3. Click **Browse** → select your photos  
4. Pick **Output Format** (JPEG, PNG, BMP, WebP)  
5. Choose **Output Path** (or keep Desktop)  
6. Hit **Start Conversion**  
7. Watch the blue progress bar fill → **Done!**
